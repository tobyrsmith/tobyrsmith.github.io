This is a collection of all my old exams. These are just the exam questions. There are no answer keys. You have to figure out the answers for yourselves. If you want some guidance on how to answer questions on these exams checkout the Astronomy 150 test taking tutorial.

Some of the questions I still use as the basis for new questions, some have been superseded by new spacecraft data, so will not be asked in the same way, and some should have never been asked in the first place. Use them at your own risk.

Remember, since I do not (usually) repeat questions, these questions are exactly the ones that will not be on your exam. Do NOT memorize the answers to these questions, that path only leads to pain.
